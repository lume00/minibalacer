# Minibalancer
*Minibalancer* is a light and simple HTTP load balancer.

**Features**:
- HTTP protocols:  HTTP/1.1, HTTP/2 and HTTP/3.
- Load balancing algorithms: Round-Robin, Failover.
- Virtual Host
- Proxy Pass
- Stateless persistent session
- SNI (Server Name Indication)
